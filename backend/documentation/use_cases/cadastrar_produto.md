# Caso de Uso: Cadastrar Produto

| Item             | Descrição                                                                 |
|------------------|--------------------------------------------------------------------------|
| Nome             | Cadastrar Produto                                                        |
| Ator             | Lojista                                                                  |
| Pré-condições    | Lojista autenticado                                                      |
| Fluxo Principal  | 1. Lojista acessa área de produtos<br>2. Informa dados do produto<br>3. Sistema salva produto |
